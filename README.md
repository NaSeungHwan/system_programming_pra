# 시스템 프로그래밍 학습을 위한 레포지터리

그동안 OS 를 사용하지 않거나 RTOS 만을 이용해 쓰래드 환경을 이용했는데, 임베디드 리눅스와 같은 환경에 대한 준비가 필요함을 느꼈다(그리고 망할 군대 대비,,,). 레포지터리에 예제용 코드 등을 같이 업로드 하면 좋겠다고 생각했기 때문에 위키와 예제 소스를 정리할 예정이다.
